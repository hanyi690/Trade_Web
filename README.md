## 2. 技术栈

本系统采用典型的分层架构开发 ：

* 
**开发语言**: Java 24 


* 
**核心框架**: Spring Boot 3.2.4 (Spring MVC, Spring Data JPA, Spring Security) 


* 
**数据库**: MySQL 8.0 


* 
**前端技术**: JSP, JSTL, Bootstrap 5 


* 
**构建工具**: Maven 3.8+ 


* 
**部署环境**: Docker, Ubuntu 22.04 LTS 



## 3. 核心功能模块

### 3.1 顾客功能

* 
**用户管理**: 注册、登录、注销及个人信息编辑 。


* 
**购买流程**: 浏览/查询产品 -> 加入购物车 -> 订单结算 -> 模拟支付 -> 异步邮件确认收货 。


* 
**订单追踪**: 查看订单状态及个人历史订单记录 。



### 3.2 商家管理

* 
**商品目录管理**: 实现商品的增删改查、上架/下架及库存管理 。


* 
**订单处理**: 商家可执行订单确认与发货操作 。


* 
**销售统计**: 后台数据可视化展示，包括今日销量、历史订单数及销售趋势折线图 。



### 3.3 平台管理员

* 
**全局管理**: 负责全平台用户、商铺及商品的监督管理 。


* 
**统计报表**: 查看全平台总流水、总用户数等聚合统计指标 。



## 4. 系统设计要点

* 
**安全性**: 基于 Spring Security 实现 RBAC（基于角色的访问控制），区分管理员、商家与顾客权限 。


* 
**事务管理**: 订单创建流程使用 `@Transactional` 保证库存扣减与订单生成的一致性 。


* 
**异步通知**: 支付成功后通过异步邮件服务（SSL/465端口）发送确认邮件，不阻塞主线程 。


* 
**查询优化**: 在 Repository 层利用 JPQL 聚合查询（如 `SUM` 销售额），提升大数据量下的统计性能 。



## 5. 安装与部署

系统支持 Docker 一键部署 ：

### 5.1 环境准备

确保已安装 Docker 和 Docker Compose 。

### 5.2 部署步骤

1. **克隆项目**:
```bash
git clone https://github.com/hanyi690/eshop

```


2. **构建并启动**:
在项目根目录下，执行以下命令：
```bash
docker-compose up --build -d

```



该命令会同时启动 MySQL 数据库容器及 Web 应用容器 。



### 5.3 环境变量配置

可在 `docker-compose.yml` 中修改以下关键环境变量 ：

* `DB_PASS`: 数据库密码
* `MAIL_USER` / `MAIL_PASS`: 邮件服务账号及授权码
* 
`APP_BASE_URL`: 部署后的域名或 IP
